⚙️ GitHub Sync Settings

Step 1 — Create a private GitHub repo (one person does this once):
1. Go to github.com → New repository → name it e.g. outage-data → Private → Create
2. Add a file named outage-data.json in the repo (content: {})
3. Share the repo name, owner, and branch with all collaborators

Step 2 — Create one shared Bot PAT (repo owner does this once):
1. GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new
2. Name it e.g. Outage Plan App, set expiry, tick scope: repo → Generate & copy
3. Share the token with your team — each person pastes it above and clicks Save & Connect

Every collaborator enters the same Owner / Repo / Branch / File Path / Token. Token is saved in their browser — entered once only.
Data auto-syncs every 30 s. No Azure. No Microsoft account required.
